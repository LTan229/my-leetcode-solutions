- 链表是一种通过指针串联的线性结构
- 节点包括数据和指向下一节点的指针
- 类型：单链表、双链表（包含前向指针）、循环链表
- 删除节点：前一个指向下一个
- 添加节点：前一个指向新
- 性能：

|     | 增删  | 查询  |
| --- | --- | --- |
| 数组  | On  | O1  |
| 链表  | O1  | On  |
```python
class ListNode:
    def __init__(self, val, next=None):
        self.val = val
        self.next = next
```
