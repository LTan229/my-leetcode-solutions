
| **数据规模 N (最大值)**  | **目标时间复杂度**          | **常见算法联想**                                                               |
| ----------------- | -------------------- | ------------------------------------------------------------------------ |
| **$N \le 20$**    | $O(2^N)$ 或 $O(N!)$   | **DFS / 回溯** (全排列、子集生成)、**状态压缩 DP** (位掩码)                                |
| **$N \le 100$**   | $O(N^3)$ 或 $O(N^4)$  | **Floyd 算法** (多源最短路)、**区间 DP**、高维 DP                                     |
| **$N \le 500$**   | $O(N^3)$ (勉强)        | 简单的 $N^3$ 模拟，矩阵乘法                                                        |
| **$N \le 2,000$** | $O(N^2)$             | **冒泡/选择/插入排序**、**基础 DP** (两层循环)、**任意两点最短路** (朴素 Dijkstra / Bellman-Ford) |
| **$N \le 10^4$**  | $O(N \sqrt{N})$      | 莫队算法 (Mo's algorithm)、分块算法 (较少见)                                         |
| **$N \le 10^5$**  | $O(N \log N)$        | **排序** (快排/归并)、**二分查找** (多次)、**堆 (Heap)**、**线段树/树状数组**、**Dijkstra+堆**    |
| **$N \le 10^6$**  | $O(N)$               | **双指针**、**滑动窗口**、**KMP**、**单调栈/队列**、**哈希表**、**并查集**、**贪心**               |
| **$N \ge 10^9$**  | $O(\log N)$ 或 $O(1)$ | **数学推导**、**二分查找** (一次)、**矩阵快速幂**、**最大公约数 (GCD)**                         |
