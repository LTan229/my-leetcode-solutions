# 哈希表
```python
from collections import Counter

class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        magcnt = Counter(magazine)
        notecnt = Counter(ransomNote)
        for k in notecnt:
            if magcnt[k] < notecnt[k]:
                return False
        return True
```
- 只做一个dict，另一个字符串去dict里减
- 用list做哈希表，反正只有26个字母
