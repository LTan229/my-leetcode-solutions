```python
class Solution:
    def reverseWords(self, s: str) -> str:
        sarr = s.strip().split(' ')
        result = ""
        for substr in sarr[::-1]:
            if len(substr) > 0:
                result += substr + " "
        return result[:-1]
```

- 移除多余空格
- 将整个字符串反转（同时移除空格）
- 将每个单词反转
