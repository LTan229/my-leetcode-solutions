- 因为是乘积，所以会负负得正，所以 dp要记录以 i - 1 结尾的子序列的最大和最小乘积
- Mindp[i] = min(arr[i], maxdp[i - 1] * arr[i], mindp[i - 1] * arr[i])
- Maxdp[i] = max(arr[i], maxdp[i - 1] * arr[i], mindp[i - 1] * arr[i])
- Return max(maxdp)
- Dp 不一定要记录全局最大，也可以记录局部最大。
	- 局部最大：连续子数组
	- 全局最大：非连续
- 什么时候记录以 i 结尾的序列，什么时候记录 i 之前的序列（可能不包含 i）
	- 包含第 i 个元素：连续子数组
	- 不包含：非连续
- 双指针只能在单调序列上（比如：加一个正数，和一定变大；减一个正数，和一定变小）
- 前缀和会导致零除

# DP
```python
def maxProduct(nums):
    if not nums: return 0
    
    # 初始化为第一个元素
    res = cur_max = cur_min = nums[0]
    
    for i in range(1, len(nums)):
        n = nums[i]
        
        # 如果当前数是负数，最大和最小值交换
        # 因为：最大值 * 负数 -> 最小值；最小值 * 负数 -> 最大值
        if n < 0:
            cur_max, cur_min = cur_min, cur_max
            
        # 核心状态转移
        cur_max = max(n, cur_max * n)
        cur_min = min(n, cur_min * n)
        
        # 更新全局答案
        res = max(res, cur_max)
        
    return res
```

# 前缀积（模拟）
- 两个方向求出数组开头/结尾到每个位置的积，但遇到 0 就重新计算
	- 遇到0就重置其实用0给数组分段了。那么每个位置的意义就是两个方向以这个位置结尾的子数组的积。然后求这个积的最大值就可以
- 对于任何一个连续子数组，如果它的乘积是最大的：
- 如果有偶数个负数：整个区间的乘积就是最大，前缀积扫到末尾时能抓到它。
- 如果有奇数个负数：最大乘积一定是“去掉最左边那个负数及其左侧”或者“去掉最右边那个负数及其右侧”。
```python
def maxProduct(nums):
    # 正向扫一遍，反向扫一遍
    reverse_nums = nums[::-1]
    
    for i in range(1, len(nums)):
        # 遇到 0 就把积重置为 1（相当于断开重新开始），否则就累乘
        nums[i] *= nums[i - 1] or 1
        reverse_nums[i] *= reverse_nums[i - 1] or 1
        
    return max(nums + reverse_nums) # concat，返回两个数组中最大的值
```
