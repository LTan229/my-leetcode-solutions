# 组合问题
```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        vstep = m - 1
        hstep = n - 1
        result = 1
        for i in range(1, m):
            result *= (i + hstep) / i
        return round(result)

# m-1 + n-1 selkect m-1
# (m-1+n-1)!/(n-1)!(m-1)!
```

# 动态规划
```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[0] * n for _ in range(m)]
        dp[m - 1].[n - 1] = 1
        for i in range(m - 1, -1, -1):
            for j in range(n - 1, -1, -1):
                if i == m - 1 and j == n - 1:
                    continue
                downcnt = dp[i + 1].[j] if i < m - 1 else 0
                rightcnt = dp[i].[j + 1] if j < n - 1 else 0
                dp[i].[j] = downcnt + rightcnt 
        return dp[0].[0]
```

1. 确定dp数组（dp table）以及下标的含义
	1. dp[i].[j] ：表示从（0 ，0）出发，到(i, j) 有dp[i].[j]条不同的路径。
2. 确定递推公式
	1. dp[i].[j] = dp[i - 1].[j] + dp[i].[j - 1]，因为dp[i].[j]只有这两个方向过来。
3. dp数组的初始化
	1. dp[i].[0]一定都是1，因为从(0, 0)的位置到(i, 0)的路径只有一条，那么dp[0].[j]也同理。
4. 确定遍历顺序
	1. 这里要看一下递推公式dp[i].[j] = dp[i - 1].[j] + dp[i].[j - 1]，dp[i].[j]都是从其上方和左方推导而来，那么从左到右一层一层遍历就可以了。
