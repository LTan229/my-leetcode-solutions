```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    cnt = 0
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        if head is None:
            return head
        rmed = self.removeNthFromEnd(head.next, n)
        head.next = rmed
        self.cnt += 1
        if self.cnt == n:
            return rmed
        else:
            return head
```

# 快慢指针（一次遍历）
使用两个指针，fast 和 slow。
设置虚拟头节点 (Dummy Node)：创建一个 dummy 节点，让它的 next 指向真正的 head。这样做是为了轻松处理“删除的就是头节点”的边缘情况（例如 [1], $n=1$）。
拉开差距：让 fast 指针先从 dummy 节点走 $n$ 步。
同时前进：现在 fast 和 slow 之间有了 $n$ 个节点的差距。同时移动 fast 和 slow 指针，一步一个节点，直到 fast 指针到达链表的末尾（即 fast.next 为 None）。
定位与删除：当 fast 到达末尾时，slow 指针停留的位置恰好是待删除节点的前一个节点。
执行删除操作：slow.next = slow.next.next。返回：返回 dummy.next（这可能是原来的 head，也可能是 head.next）。
