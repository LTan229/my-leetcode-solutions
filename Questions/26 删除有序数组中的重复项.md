
- 非严格递增排列：后面的数可以等于前面的数，但整体是递增

我的解法：-1有越界风险
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        fast, slow = 0, 0
        len_nums = len(nums)
        cur_num = nums[0] - 1
        while fast < len_nums:
            if nums[fast] > cur_num:
                nums[slow] = nums[fast]
                cur_num = nums[slow]
                slow += 1
            fast += 1
        return slow
```

题解：
```python
class Solution: 
	def removeDuplicates(self, nums: List[int]) -> int: 
	n = len(nums) 
	fast = slow = 1 # 留下0
	while fast < n: 
		if nums[fast] != nums[fast - 1]: # 如果fast遇到了新的
			nums[slow] = nums[fast] 
			slow += 1 
		fast += 1 
	return slow
```
