- 滑动窗口：用两根指针维护一个窗口，如果窗口值大于目标了，就收缩左指针，如果窗口值小于目标了，就往前推右指针。停止条件是右指针超过数组大小。能实现O(n)
```python
class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        start, end = 0, 0
        sumval = nums[0]
        len_nums = len(nums)
        minlen = len(nums)
        reachflag = False
        while True:
            if sumval < target:
                end += 1
                if end >= len_nums:
                    break
                sumval += nums[end]
            else:
                reachflag = True
                
                curlen = end - start + 1
                minlen = curlen if curlen < minlen else minlen
                sumval -= nums[start]
                start += 1
        
        return minlen if reachflag else 0

```
- 前缀和+二分查找：首先计算数组前n项和。然后寻找两项和之间的差，使其不低于target，那么其实问题就转换为，对于每一个前n项和，向后找大于target+前n项和的k项和。如果找到则更新最小长度。（python不会溢出但是别的语言有溢出风险）
```python
class Solution:
    def minSubArrayLen(self, s: int, nums: List[int]) -> int:
        if not nums:
            return 0
        
        n = len(nums)
        ans = n + 1
        sums = [0]
        for i in range(n):
            sums.append(sums[-1] + nums[i])
        
        for i in range(1, n + 1):
            target = s + sums[i - 1]
            bound = bisect.bisect_left(sums, target)
            if bound != len(sums):
                ans = min(ans, bound - (i - 1))
        
        return 0 if ans == n + 1 else ans
```
