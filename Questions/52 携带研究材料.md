#  朴素想法
在01背包的基础上加一维尝试不同数量
超时
```python
n, cap = map(int, input().split())
ws = []
vs = []
for _ in range(n):
    w, v = map(int, input().split())
    ws.append(w)
    vs.append(v)

dp = [0] * (cap + 1)
for i in range(n): # ith item
    for j in range(cap, ws[i] - 1, -1):# capacity
        maxval = 0
        for k in range(0, j // ws[i] + 1): # select cnt
            curval = k * vs[i] + dp[j - k * ws[i]]
            maxval = max(maxval, curval)
        dp[j] = maxval

print(dp[-1])
```

# 正确解法
正序遍历dp数组
这样修改更大重量格子的时候相当于放多个物品，而且已经考虑到了放比较少数量的物品的情况
```python
n, cap = map(int, input().split())
ws = []
vs = []
for _ in range(n):
    w, v = map(int, input().split())
    ws.append(w)
    vs.append(v)

dp = [0] * (cap + 1)
for i in range(n): # ith item
    for j in range(ws[i], cap + 1):# capacity
            dp[j] = max(dp[j], vs[i] + dp[j - ws[i]])

print(dp[-1])
```
