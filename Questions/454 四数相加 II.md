# 哈希
首先遍历AB的所有组合，储存在哈希表里，然后遍历CD的所有组合，去哈希表里找匹配的组合
```python
class Solution:
    def fourSumCount(self, nums1: List[int], nums2: List[int], nums3: List[int], nums4: List[int]) -> int:
        setab = dict()
        for n1 in nums1:
            for n2 in nums2:
                t = n1 + n2
                if t in setab:
                    setab[t] += 1
                else:
                    setab[t] = 1
        result = 0
        for n3 in nums3:
            for n4 in nums4:
                t = n3 + n4
                if -t in setab:
                    result += setab[-t]
        
        return result
```
