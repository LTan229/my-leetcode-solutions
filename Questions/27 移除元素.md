- 双指针法
	- fast用来找下一个有效的数字
	- slow则用来维护所有有效数字的队列
	- slow走过的要么是val（无效数字），要么是已经被移动走的空白位置。所以这个方法是完备的

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        fast, slow = 0, 0
        len_nums = len(nums)
        while fast < len_nums:
            if nums[fast] != val:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        return slow
```
