```python
class Solution:
    def findLength(self, nums1: List[int], nums2: List[int]) -> int:
        len1 = len(nums1)
        len2 = len(nums2)
        # 以 nums1[i] nums2[j] 结尾的最长公共子数组
        dp = [[0] * len2 for _ in range(len1)]
        for i in range(len1):
            for j in range(len2):
                if nums1[i] != nums2[j]:
                    continue
                if i > 0 and j > 0:
                    dp[i][j] = dp[i - 1][j - 1] + 1  
                else:
                    dp[i][j] = 1
        return max([max(row) for row in dp])
```
- 令 `dp[i][j]` 表示： **“以 `nums1[i]` 和 `nums2[j]` 这两个元素结尾的、公共的、最长子数组的长度。”**
- 如果当前的两个数字都不一样，那以它们结尾的公共子数组长度肯定是 **0**。
- **这两个数字相等 (`nums1[i] == nums2[j]`)** ，回头看**紧挨着的上一个位置**，也就是 `nums1[i-1]` 和 `nums2[j-1]`。
