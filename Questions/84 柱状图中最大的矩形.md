```python
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        if len(heights) == 1:
            return heights[0]
        heights = [0] + heights + [0] # left 0: prevent empty stack, avoid checking, unify w calculation
        # right 0: make sure the stack is checked after the last piece is added
        stack = [0]
        area = 0
        for i in range(1, len(heights)):
            while stack and heights[i] < heights[stack[-1]]:
                pop_idx = stack.pop()
                w = i - stack[-1] - 1 # do not count i and stack[-1]
                h = heights[pop_idx] # area between i and stack[-1], column i calculated in future rounds
                area = max(area, w * h)
            stack.append(i)
        return area

```
- find the nearest bins on the left and right side that is smaller than the current bins O N
- 双指针：左右两个指针，选中更高的那个。指针外有左右历史。如果右指针<右历史说明历史不作数，更新历史，删除已经记录的面积，否则开始统计面积
- 单调栈。如果下一个元素大于栈顶，则入栈，否则出栈直到栈顶元素小于下一个元素。每出一个栈用出栈元素的高度计算一次栈顶、下一个元素之间的矩形距离。
- 还有一个思路就是。对于第一个柱子，最大面积是它本身。对于每个新柱子，考虑i-1个柱子的时候矩形是否挨着这个柱子，如果挨着，就考虑是否使用这个柱子。使用：面积=min（原始矩形高度，新柱子高度） * （原始矩形宽度 + 1）。不使用：面积=原始矩形。这两个数字都记录下来给添加下一个柱子的情况选择。

# 双指针

```python
class Solution:
    def largestRectangleArea(self, heights: List[int]) -> int:
        size = len(heights)
        # 两个DP数列储存的均是下标index
        min_left_index = [0] * size
        min_right_index = [0] * size
        result = 0

        # 记录每个柱子的左侧第一个矮一级的柱子的下标
        min_left_index[0] = -1  # 初始化防止while死循环
        for i in range(1, size):
            # 以当前柱子为主心骨，向左迭代寻找次级柱子
            temp = i - 1
            while temp >= 0 and heights[temp] >= heights[i]:
                # 当左侧的柱子持续较高时，尝试这个高柱子自己的次级柱子（DP
                temp = min_left_index[temp]
            # 当找到左侧矮一级的目标柱子时
            min_left_index[i] = temp
        
        # 记录每个柱子的右侧第一个矮一级的柱子的下标
        min_right_index[size-1] = size  # 初始化防止while死循环
        for i in range(size-2, -1, -1):
            # 以当前柱子为主心骨，向右迭代寻找次级柱子
            temp = i + 1
            while temp < size and heights[temp] >= heights[i]:
                # 当右侧的柱子持续较高时，尝试这个高柱子自己的次级柱子（DP
                temp = min_right_index[temp]
            # 当找到右侧矮一级的目标柱子时
            min_right_index[i] = temp
        
        for i in range(size):
            area = heights[i] * (min_right_index[i] - min_left_index[i] - 1)
            result = max(area, result)
        
        return result
```
