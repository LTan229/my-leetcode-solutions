# Stack
```python
class Solution:
    def removeDuplicates(self, s: str) -> str:
        q = []
        for idx, char in enumerate(s):
            if q and q[-1].startswith(char):
                q.pop()
            else:
                q.append(char)
            
        return "".join(q)
```
