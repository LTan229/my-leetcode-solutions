```python
class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        n = len(cost)
        dp = [0, 0]
        for i in range(1, n):
            dp.append(min(dp[i + 1 - 2] + cost[i + 1 - 2], dp[i + 1 - 1] + cost[i + 1 - 1]))
        
        return dp[-1]


# 10, 15, 20

#  0,  0   ?      ?

# 1, 100
#     i
# 0,  0  ?
```
1. 确定dp数组以及下标的含义
	1. 到达第i台阶所花费的最少体力为dp[i]
2. 确定递推公式
	1. dp[i] = min(dp[i - 1] + cost[i - 1], dp[i - 2] + cost[i - 2]);
3. dp数组如何初始化
	1. 题目描述中明确说了 “你可以选择从下标为 0 或下标为 1 的台阶开始爬楼梯。” 也就是说 到达 第 0 个台阶是不花费的，但从 第0 个台阶 往上跳的话，需要花费 cost[0]。
4. 因为是模拟台阶，而且dp[i]由dp[i-1]dp[i-2]推出，所以是从前到后遍历cost数组就可以了。
