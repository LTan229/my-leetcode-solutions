动态规划
- `grid[i][j][k] = m`，表示 **节点i 到 节点j 只经过[1...k] 集合中的节点的最短距离为m**。
- 递推公式`grid[i][j][k] = min(grid[i][k][k - 1] + grid[k][j][k - 1]， grid[i][j][k - 1])
	- 节点i 到 节点j 的最短路径经过节点k
		- `grid[i][j][k] = grid[i][k][k - 1] + grid[k][j][k - 1]`
			- 节点i 到 节点k 的最短距离 是不经过节点k，中间节点集合为[1...k-1]，所以 表示为`grid[i][k][k - 1]`
			- 节点k 到 节点j 的最短距离 也是不经过节点k，中间节点集合为[1...k-1]，所以表示为 `grid[k][j][k - 1]`
	- 节点i 到 节点j 的最短路径不经过节点k
		- `grid[i][j][k] = grid[i][j][k - 1]`
			- 如果节点i 到 节点j的最短距离 不经过节点k，那么 中间节点集合[1...k-1]，表示为 `grid[i][j][k - 1]`
- 初始化：
	- 对于边p1, p2, val, 初始化
		- `grid[p1][p2][0] = val;`
		- `grid[p2][p1][0] = val;`
	- 其它`grid[p1][p2][0]`初始化为最大值
- 遍历顺序
	- 需要三个for循环，分别遍历i，j 和k
	- k在最外面，因为`grid[p1][p2][k]`依赖于`grid[p1][p2][k-1]`
# 只用一层
```python
n, m = map(int, input().split())

matrix = [[float('inf') for _ in range(n + 1)] for __ in range(n + 1)]

for _ in range(m):
    u, v, w = map(int, input().split())
    matrix[u][v] = w
    matrix[v][u] = w

for k in range(1, n + 1):
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            matrix[i][j] = min(
                matrix[i][k] + matrix[k][j],
                matrix[i][j]
            )

q = int(input())

for _ in range(q):
    start, end = map(int, input().split())
    if matrix[start][end] == float('inf'):
        print(-1)
    else:
        print(matrix[start][end])
```

# 用两层

```python
n, m = map(int, input().split())

matrix = [[[float('inf')] * 2 for _ in range(n + 1)] for __ in range(n + 1)]

for _ in range(m):
    u, v, w = map(int, input().split())
    matrix[u][v][0] = w
    matrix[v][u][0] = w

for k in range(1, n + 1):
    cur = k % 2
    prev = (k - 1) % 2
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            matrix[i][j][cur] = min(
                matrix[i][k][prev] + matrix[k][j][prev],
                matrix[i][j][prev]
            )

q = int(input())

for _ in range(q):
    start, end = map(int, input().split())
    if matrix[start][end][n % 2] == float('inf'):
        print(-1)
    else:
        print(matrix[start][end][n % 2])
```
