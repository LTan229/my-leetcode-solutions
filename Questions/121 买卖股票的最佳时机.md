```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        dp = [[-prices[0],0]]
        for i in range(1,len(prices)):
            wval = max(dp[i - 1][0], - prices[i])
            woval = max(dp[i - 1][1], dp[i - 1][0] + prices[i])
            dp.append([wval, woval])
        return max(dp[-1])
```

- DP数组
	- `dp[i][0]` 表示如果第i天持有股票持有现金
		- 第i-1天就持有股票，持有现金第i-1天现金：`dp[i - 1][0]`
		- 第i天买入股票，持有现金就是（用所有本金）买入今天的股票后所得现金即：`0 -prices[i]`
	- `dp[i][1]` 表示如果第i天不持有股票持有最多现金
		- 第i-1天就不持有股票，那么持有现金是第i-1天现金`dp[i - 1][1]`
		- 第i-1天持有股票，第i天卖了，持有现金是`dp[i - 1][0] + price[i]`
	- `dp[i][0] = max(dp[i - 1][0], - prices[i]);` 和` dp[i][1] = max(dp[i - 1][1], prices[i] + dp[i - 1][0]);`
	- $dp[i][0]$ 其实一直在记录 **历史上出现过的“最大负数”**（也就是绝对值最小的价格，即最低买入成本）。
		- 过去成本和现在成本之间的最小值
	- $dp[i][1]$ 其实一直在记录 **当前能锁定的最大利润**。
		- 过去利润和现在利润之间的最大值
