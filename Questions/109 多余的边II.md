```python
n = int(input())

edge = []
degree = [[0, 0] for _ in range(n + 1)] # in, out

twoin = 0
twoout = 0
for _ in range(n):
    s, t = map(int, input().split())
    edge.append((s, t))
    degree[s][1] += 1
    degree[t][0] += 1
    if degree[s][1] == 2:
        twoout = s 
    if degree[t][0] == 2:
        twoin = t 

parent = [i for i in range(n + 1)]

def find(u):
    if parent[u] == u:
        return u
    else:
        parent[u] = find(parent[u])
        return parent[u]

# print(twoout, twoin)
if twoin != 0: # there exist a node where in degree is 2
    edge1 = None
    edge2 = None
    for s, t in edge:
        if t == twoin: # store the first edge, skip the second
            if not edge1:
                edge1 = (s, t)
            else:
                edge2 = (s, t)
                continue
        ps = find(s)
        pt = find(t)
        if ps == pt: # if is circle
            print(edge1[0], edge1[1])
            exit()
        else: # no circle
            parent[pt] = ps
    # no circle without edge2
    print(edge2[0], edge2[1])
    exit()
else: # there is a circle, try to delete the last edge that creates a circle
    for s, t in edge:
        ps = find(s)
        pt = find(t)
        if ps == pt: # if is circle
            print(s, t)
            exit()
        else: # no circle
            parent[pt] = ps
```

1. 遍历所有的边，找到入度出度为2的节点
2. 如果存在入度为2的节点，说明两条入边里有一条多余。
	1. 删除edge2，构建并查集，若成环，则说明edge1多余，否则edge2多余
3. 不存在入度为2的节点，则是普通的并查集判环情况，遇到的第一条导致成环的边就是我们要删除出的边。
