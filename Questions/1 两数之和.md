# 哈希表（映射）
需要快速把新元素和旧元素进行比对（哈希表），并且保存旧元素的下标（map）
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        d = dict()
        for idx, n in enumerate(nums):
            if target - n in d:
                return [d[target - n], idx]
            d[n] = idx
```


```c++
#include <vector>
#include <unordered_map>

using namespace std;

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int, int> hashtable;
        for (int i = 0; i < nums.size(); i++) {
            int complement = target - nums[i];
            auto it = hashtable.find(complement); // auto class, must be inniciated, cant be used before c++ 14, readability

            if (it != hashtable.end()) {
                return {it->second, i};
            }

            hashtable[nums[i]] = i;
        } 
        return {};
    }
};
```
