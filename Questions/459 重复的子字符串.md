# KMP
- j指向pattern串/前缀末尾，初始化next的第0位
- i指向主串/后缀末尾，从1开始
- 分为不匹配的时候回溯j、匹配的时候增加j和实现目标三步
	- 构造next时，目标是记录j
	- 检查主串的时候，目标是判断模式串是否走完
- 这道题只需要判断前后缀不为0&剩下字符串长度可以被总长度整除
```python
class Solution:
    def repeatedSubstringPattern(self, s: str) -> bool:
        lens = len(s)
        if lens == 1:
            return False
        j = -1
        nexts = [0] * lens
        nexts[0] = j
        for i in range(1, lens):
            # while not match
            while j >= 0 and s[j + 1] != s[i]:
                j = nexts[j]
            # when match
            if s[j + 1] == s[i]:
                j += 1
            # objective
            nexts[i] = j

        lps_len = nexts[-1] + 1
        remlen = lens - lps_len
        if lens % remlen == 0 and lps_len > 0:
            return True

        return False
```

# 字符串匹配
如果将两个 s 连在一起，并移除第一个和最后一个字符，那么得到的字符串一定包含 s，即 s 是它的一个子串

```java
class Solution {
public:
    bool repeatedSubstringPattern(string s) {
        string t = s + s;
        t.erase(t.begin()); t.erase(t.end() - 1); // 掐头去尾
        if (t.find(s) != std::string::npos) return true; // r
        return false;
    }
};
```
